
# Credit Card Fraud Risk Analysis - Power BI Dashboard Project

This project focuses on analyzing credit card transaction data to identify fraudulent activities and assess transaction risk using **Power BI**. The goal is to derive meaningful insights from real-world financial data and present them through an interactive, decision-oriented dashboard.

The project demonstrates practical skills in **data cleaning, analysis, DAX, and dashboard design**, aligned with real banking and financial analytics use cases.

---

## Project Objectives

- Analyze transaction data to identify fraud patterns
- Measure fraud rate and high-risk transaction behavior
- Design an interactive dashboard for fraud monitoring
- Apply business intelligence techniques used in banking systems

---


## Key Concepts & Skills Applied

- Data Cleaning & Preparation
- Exploratory Data Analysis (EDA)
- DAX Measures & Calculations
- Fraud Rate & Risk Metrics
- Interactive Dashboard Design
- Data Visualization & Storytelling

## Dashboard Features

- Fraud rate analysis
- Transaction volume and trends
- Fraud type comparison
- High-risk transaction identification
- Interactive filters and slicers for deep analysis

---

## Tools & Technologies Used

- Power BI Desktop
- DAX
- Excel / CSV Dataset
- Data Visualization Techniques
